## 🎨 Características Técnicas

### HTML5 Semántico
- Header con navegación principal
- Etiquetas semánticas correctas (`<main>`, `<section>`, `<article>`, `<footer>`)
- Formularios con validación HTML5
- Imágenes con atributos `alt` descriptivos

### CSS3 Avanzado
- **Selectores**: etiqueta, clase, id, descendientes, hijos
- **Pseudo-clases**: `:hover`, `:active`, `:first-child`
- **Pseudo-elementos**: `::before`, `::after`
- **Flexbox**: para layouts flexibles y responsive
- **CSS Grid**: para grillas complejas de productos
- **Box Model**: margin, padding, border, border-radius, box-shadow

### Responsive Design
- **Móvil (<768px)**: Menú vertical, productos en 1 columna
- **Tablet (768-1023px)**: 2 columnas de productos
- **Desktop (≥1024px)**: CSS Grid con múltiples columnas
- **Desktop Grande (≥1440px)**: 4 columnas de productos

## 🚀 Cómo Usar

1. **Abrir el sitio**: Abre `index.html` en tu navegador
2. **Navegación**: Usa el menú principal para explorar
3. **Productos**: Explora el catálogo en `productos.html`
4. **Contacto**: Envía mensajes desde `contacto.html`
5. **Registro**: Crea una cuenta en `registro.html`
